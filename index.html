🚗 CarPlay Guard

Sileo Repository

Tự động quản lý kết nối CarPlay để tiết kiệm pin và quản lý ứng dụng

https://raw.githubusercontent.com/window8/carplay-guard/main/

✨ Tính Năng

  • ✅ Tự động bật WiFi khi CarPlay kết nối
  • ✅ Tự động bật 4G/dữ liệu di động
  • ✅ Tự động mở YouTube (tùy chọn)
  • ✅ Tự động đóng ứng dụng khi ngắt kết nối
  • ✅ Tự động tắt WiFi, Bluetooth, 4G
  • ✅ Tự động khóa màn hình

📱 Cách Cài Đặt

  1. Mở Sileo trên iPhone
  2. Đi tới tab Sources
  3. Tap + (thêm)
  4. Dán link repo:
  5. https://raw.githubusercontent.com/window8/carplay-guard/main/
  6. Tap Add Source
  7. Tìm CarPlay Guard → Tap Get → Install

⚙️ Yêu Cầu

  • iOS 15.0 hoặc cao hơn
  • iPhone 6S trở lên
  • Jailbreak với ElleKit
